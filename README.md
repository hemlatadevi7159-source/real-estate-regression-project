# Real Estate Regression Project

Minor AI Project: Real Estate Price Prediction using Linear, Multiple, and Polynomial Regression implemented from scratch.

## Features
- Simple Linear Regression
- Multiple Linear Regression
- Ridge & Lasso Regularization
- Polynomial Regression
- K-Fold Cross Validation
- Feature Selection
- VIF Analysis
- Ensemble Methods
- Performance Evaluation

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
